# GITOFFICE

Monorepo with:
- `server`: Node.js + Express + PostgreSQL API
- `client`: React + Vite + Tailwind app

## Quick Start

1. Install dependencies
   - `npm install`
2. Run server
   - `npm run dev:server`
3. Run client
   - `npm run dev:client`

## Structure

See folders in `server/` and `client/` for modular architecture.
